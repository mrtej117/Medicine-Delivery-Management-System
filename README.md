# Online Medicine Delivery Management System

A simple console-based Python project that simulates an online medicine delivery system.  
This project manages customers, products, shops, warehouses, delivery agents, inventory, and order processing using Object-Oriented Programming concepts in Python.

Built as a beginner-friendly project to practice:
- OOP Concepts
- Lists & Dictionaries
- Functions
- Menu Driven Programs
- Basic Inventory Management
- Real-time Order Simulation

---

# Features

## Customer Management
- Add new customers
- Delete customers
- Display customer details
- Track purchased products

## Product Management
- Add products
- Delete products

## Shop & Inventory Management
- Create shops
- Add inventory to shops
- View shop inventory

## Delivery Agent Management
- Add delivery agents
- Delete agents
- Automatically assign least busy agent

## Order Processing
- Place orders
- Match customer zipcode with nearby shop
- Check inventory availability
- Simulate delivery process
- Update inventory after delivery

# How It Works

1. Create customers, products, shops, and delivery agents
2. Add products to shop inventory
3. Customers place orders
4. System checks:
   - Valid customer
   - Product availability
   - Matching zipcode
5. Delivery agent gets assigned
6. Order gets delivered successfully
